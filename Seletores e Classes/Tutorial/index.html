Seletores e Classes

Seletores

São as formas de fazer o computador entender qual elemento ele deve alterar.

Tipos de Seletores:

  1. Tags: Dizer uma tag específica que você quer que o computador altere.
  2. Classes: Atribuir uma classe a uma tag, assim a tag vai ter as propriedades dadas a essa classe.
  3. ID: "Batiza" um elemento com um nome, e apenas ele terá as propriedades CSS atribuidas ao ID dele.
  4. Curinga (*): Ao usar "*" você aplica as propriedades CSS a todos os elementos da página.
  5. Agrupar (,): Ao usar "," você pode aplicar as propriedades CSS em vários elementos ao mesmo tempo, exemplo: h1, h2, h3.

Detalhes:

Nenhum nome de classe e ID pode começar com números e não podem ter acentos, e os IDs são apenas 1 ID para 1 elemento, não se pode usar 1 mesmo ID em 2 elementos por que o ID é único para apenas 1 elemento específico.
